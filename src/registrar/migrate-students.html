Migrate Students

Instructions for Uploading Students:
  1. Must be in 'XLSX' file extension.
  2. File content must follow the exact format.
  3. Must not use comma because it serves as the separator of every field.
  4. Duplicate Students will not be added.
  5. *Last Name, *First Name, *Grade Level (ex: Grade 6), Track (ex: Academic), Strand (ex: Computer Hardware Servicing (ICT)) and Birthdate (YYYY-MM-DD) are required.
  6. If any of the Columns above (#5) have blank value, the row will not be saved.
